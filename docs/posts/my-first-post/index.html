My New Hugo Site

November 30, 2018

Hugoにいく

インストール

管理者権限コマンドプロンプトで

choco install hugo

新しくつくるには

hugo new site myblog

テーマの入手と適用

テーマはこれにきめた。

https://themes.gohugo.io//theme/kiss/

cd themes
git clone https://github.com/ribice/kiss.git
cd ../
echo 'theme = "kiss"' >> config.toml

何か書く

hugo new posts/my-first-post.md

---
title: "My First Post"
date: 2018-11-30T16:23:42+09:00
draft: false
---

Hello World

表示

hugo server -D

記事の更新はホットリロードしてくれるが、config.tomlをいじった場合は再起動させる必要がある。
